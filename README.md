# ハローワーク求人まとめサイト

ハローワークインターネットサービスから抽出した最新の求人情報を、モダンで使いやすいインターフェースで閲覧できるアグリゲーターサイトです。

## 特徴
- **リアルタイム抽出**: ハローワークの検索結果HTMLから直接データを解析。
- **モダンなUI**: 直感的なデザインで、モバイル・PC両対応。
- **かんたん検索**: キーワード、勤務地、雇用枠（一般/障害者枠）で素早くフィルタリング。
- **公式連携**: ワンクリックでハローワーク公式サイトの求人詳細へアクセス可能。

## 使い方
1. `index.html` をブラウザで開きます。
2. キーワードやカテゴリーを選択して「検索」をクリック。
3. 気になる求人をクリックすると詳細が表示されます。
